# Shared Icon Backend

This directory is the future authority for the universal icon backend.

## Intended layout

- `geometry/`
  Shared master SVG geometry keyed by semantic ID.
- `skins/mate/`
  Style-layer overrides and exports for the MATE-universal skin.
- `skins/tela/`
  Style-layer overrides and exports for the Tela-universal skin.
- `aliases/`
  Name and compatibility mappings from installed filenames to semantic IDs.
- `registry/`
  Promotion outputs plus unresolved per-semantic work items for the shared
  backend intake queue.

## Contract

- Installed icon trees in `icons/Synthesis-Dark-Icons/` are outputs, not authority.
- Every backend-owned icon is keyed by a stable semantic ID.
- Shared geometry is separate from skin styling so a single concept can render in
  both universal styles without forking the source shape.
- Reports generated by `make icon-reconciliation-report` are the inventory input
  for migration into this backend.
- `registry/unresolved/<skin>/<category>/<stem>.json` is the actionable intake
  queue for unresolved semantic IDs that still need authoritative geometry.
