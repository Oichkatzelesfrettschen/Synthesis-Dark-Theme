# Requirements

## Core Dependencies
*   **GTK+ 3.x**: The primary toolkit for the theme.
*   **GTK-Engine-Murrine**: Required for GTK2 rendering.
*   **Gnome Shell**: (Optional) For the shell theme components.
*   **Metacity/Marco**: (Optional) For the window manager theme.

## Extras
*   **Tilix**: For the included terminal color scheme.

## Build Tools (For Developers)
If you intend to modify and rebuild the assets from source:
*   **Inkscape**: For rendering SVG assets.
*   **Optipng**: For optimizing generated PNGs.
*   **Node.js & npm**: For Gulp-based build workflows (inherited from Ant-Dracula).
*   **Sass/SCSS**: For compiling stylesheets.
*   **Python 3**: For execution of rendering scripts.
